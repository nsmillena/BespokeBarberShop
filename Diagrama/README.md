# Diagrama de Sequência – Agendamento de Serviço

Este diagrama de sequência representa as interações entre Cliente, Sistema e Banco de Dados durante o processo de agendamento na Bespoke Barbershop. Ele abrange desde o acesso à tela de agendamento até a confirmação final da reserva.

O objetivo do diagrama é mostrar, de forma clara e ordenada, como o sistema responde a cada ação do cliente, como escolher a unidade, selecionar o serviço, definir o barbeiro e verificar os horários disponíveis, evidenciando a comunicação entre as camadas do sistema e a troca de informações necessárias para concluir o agendamento.

Esse fluxo garante que o processo seja organizado, preciso e eficiente, permitindo que o cliente escolha todos os detalhes do atendimento e receba a confirmação do horário escolhido.
