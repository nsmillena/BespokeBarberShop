# Diagrama de Atividade – Agendamento de Serviço

Este diagrama de atividade representa o fluxo completo do processo de agendamento de serviços dentro do sistema de controle de barbearia. Ele abrange desde o momento em que o cliente realiza o login até a confirmação do agendamento, incluindo as interações com o sistema e com o barbeiro.

O objetivo do diagrama é ilustrar de forma clara e estruturada as etapas envolvidas no agendamento, destacando as decisões que o sistema toma (como validação de login e verificação de disponibilidade de horário), bem como as ações executadas por cada ator envolvido: Cliente, Sistema e Barbeiro.

Este fluxo é essencial para garantir que o agendamento ocorra de forma segura, organizada e eficiente, permitindo que o cliente escolha o serviço desejado, selecione o profissional, defina data e horário, e receba a confirmação da reserva.
